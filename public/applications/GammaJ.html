Gamma ramp editor

   GammaJ is a free color adjustment tool for monitors. It allows to set up own gamma ratio, contrast, brightness and color temperature. It is possible to achieve the effect of "gaming glasses" in few easy steps. RGB channels can be separated and inverted. Aside from assisted configuration (through preset functions connected with user input) there is also a possibility of manual color adjustment. It allows to draw ramp directly on canvas using mouse or input direct ramp values through ramp table.

Functionalities

  • Color correction:
    • Assisted adjustment of:
      • Gamma
      • Contrast "unilateral"
      • Contrast (2) "bilateral"
      • Brightness
      • Temperature
      • sRGB Temperature correction
    • Manual adjustment of gamma ramp through:
      • Drawing ramp on canvas
      • Table ramp input
    • Color inversion
    • Separable RGB channels
  • Customized color profiles
  • Multiple monitor support
  • Global hotkeys
  • System tray

Installation

   GammaJ requires system restart after first run to enable customized gamma ramps.

Device gamma ramp

   Device gamma ramp consists of 3x256 (number of RGB channels x amount of values for channel) DAC (digital-analog-converter) values in the range of 0 to 65535. In other words, gamma ramp contains mappings between RGB values in computer memory and digital-analog-converter (DAC) values on the basis of which a specific color is displayed on the monitor. Therefore, input in the table from manual color adjustment should be in the range of 0 to 65535. The default gamma ramp contains linear values.

Color profiles

   Custom gamma ramps both for assisted and manual adjustment can be saved to .properties file. All files with .properties extension (color profiles) from the same directory as GammaJ.exe will be loaded upon application start.

Hotkeys

   GammaJ allows user to define own hotkeys for changing between color profiles. To let user change color profile even when the application has neither focus nor is in tray hotkeys are registered as "global". This means that other applications will not receive input when the hotkey is pressed (even when focused). Thus, it is recommended to save color profiles with uncommon hotkeys instead of basic (A/B/C/etc.) keys. By default CTRL+SHIFT+R is registered as reset key in case of loading or setting nondistinc gamma settings and can be changed in application settings.

Troubleshooting

   By default GammaJ opens recently used profiles for each display. If an indistinct color profile is loaded and you dont remember the reset hotkey you can press alt+f4 (exit the application) and delete either "indistinct_profile".properties or (if you don't know which profile it is) delete GammaJ.ini (which also contains other settings). As a last resort (when you can't see anything) you can always restart operating system and display colors will be back to normal.

Fully customized gamma ramps require a system registry entry which by default should be added in the process of installation. After installation a system restart is needed in order for it to load new entry. Not all direct color video boards support loadable gamma ramps in hardware.
